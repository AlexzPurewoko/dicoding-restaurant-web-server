Get All Books

Mendapatkan daftar buku

  • URL

    /list

  • Method:

    GET

Add new Book

Menambahkan data buku

  • URL

    /add

  • Method:

    POST

  • Headers

    • Content-Type: application/x-www-form-urlencoded | application/json
    • X-Auth-Token: YOUR_API_KEY (for test using 12345)

  • Body

    Pilih salah satu

    • urlencoded :
      • id : number
      • title : string

      • author string

    • JSON: {"id" : number, "title" : string, "author" : string }
